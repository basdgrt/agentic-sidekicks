# Sentence-Craft Integration

## Overview

LinkedIn content must be crisp, scannable, and engaging. The sentence-craft skill provides systematic review to strengthen writing at the sentence level. Use it during the refinement step to identify and fix weak sentences.

## When to Use Sentence-Craft

Integrate sentence-craft review during **Step 2: Refine** of the LinkedIn content workflow.

After creating a draft, use the sentence-craft skill to:
- Eliminate weak constructions that dilute impact
- Strengthen word choice for precision and confidence
- Improve rhythm for LinkedIn's scanning behavior
- Remove unnecessary words that weaken the message

## How to Integrate

During the refinement step:

1. **Run sentence-craft review** on the draft
2. **Focus on high-impact issues**:
   - Weak verb constructions (e.g., "make a decision" → "decide")
   - Filler words and hedging language that weakens authority
   - Sentence rhythm that doesn't match LinkedIn's scan-and-scroll pattern
   - Word choice that's too generic or too academic
3. **Apply improvements selectively**:
   - Prioritize changes that strengthen clarity and confidence
   - Preserve the conversational, peer-to-peer tone
   - Keep technical precision without being academic
4. **Verify LinkedIn-specific requirements**:
   - Short paragraphs maintained (1-3 sentences max)
   - No em dashes (sentence-craft may suggest them - reject these)
   - Technical terms stay precise (don't simplify for general audience)

## LinkedIn-Specific Considerations

When applying sentence-craft improvements, adapt recommendations for LinkedIn:

**Paragraph length**: LinkedIn content needs frequent white space. Even if sentence-craft suggests combining sentences, keep paragraphs short for visual scanning.

**Em dashes**: Sentence-craft may suggest em dashes for flow. Replace with periods or commas for LinkedIn. Break into separate sentences if needed.

**Formality level**: LinkedIn technical content sits between conversational and professional. Accept "you" and "I" pronouns. Avoid academic distance.

**Sentence length variation**: Sentence-craft emphasizes rhythm through varied length. For LinkedIn, bias toward shorter sentences. Use very short sentences (3-5 words) for emphasis.

**Technical precision**: When sentence-craft suggests simpler words, preserve technical terms that signal expertise to the target audience (senior engineers). "Cooperative multitasking" stays, "make code better" becomes "improve performance."

## Example Integration

**Original draft**:
> AI tools are making it possible to scaffold features more quickly than before. However, there is a risk that we might lose sight of the fact that we need to make sure we understand what the code does. We should be checking tests and making certain the solution is not overly complex.

**After sentence-craft review** (identifies weak constructions, wordiness, hedging):
> AI tools scaffold features in minutes. But speed creates a risk: shipping code we don't fully understand. Before merging agent-generated code, check three things: Can you explain the solution? Do tests cover unhappy paths? Is it simpler than it needs to be?

**Key improvements**:
- "making it possible to scaffold" → "scaffold" (weak verb eliminated)
- "more quickly than before" → "in minutes" (concrete, eliminates redundancy)
- "there is a risk that we might" → "speed creates a risk" (eliminates hedging, adds concrete subject)
- "making certain the solution is not overly complex" → "Is it simpler than it needs to be?" (direct question, peer-to-peer tone)
- Multiple short sentences instead of one long paragraph

## Integration Workflow

```
1. Create draft (Step 1)
2. Review with sentence-craft skill
3. Apply high-impact improvements
4. Verify LinkedIn-specific requirements maintained
5. Proceed to quality review (Step 3)
```

The goal is not to over-polish. Apply sentence-craft improvements that strengthen confidence, clarity, and precision while preserving the authentic, peer-to-peer voice that works on LinkedIn.
